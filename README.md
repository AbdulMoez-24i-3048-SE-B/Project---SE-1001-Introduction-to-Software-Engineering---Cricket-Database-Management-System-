## Description
Our database is designed to store, edit and display certain information (shown in diagram) for teams and cricket players. The system will also allow us to generate summaries of each player to show their performance in the last 5 matches.
## Project Statement
- **The problem of:** Retrieving and displaying cricketers’ info.  
- **Affects:** The cricket players, overseeing cricket committee.  
- **The impact of which is:** Requiring a database for storing and accessing cricketers’ data.  
- **A successful solution would be:** A database designed to store and display data of each cricket team, its members and the statistics of each player, along with providing quick summaries.
